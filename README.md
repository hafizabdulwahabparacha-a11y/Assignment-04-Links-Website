# Assignment-04-Links-Website
Modern Web Application Development
Task Details:
Create an index.html file that will serve as the homepage.
Make a pages folder and create separate files for About, Services, and Contact.
In index.html, add navigation links to each of the other pages.
On each page (About, Services, Contact), create five paragraphs of text using lorem ipsum dummy content.
Assign unique id attributes to your headings and paragraphs (e.g., aboutHeadingOne, serviceParagraphTwo).
Use anchor links in index.html to connect to these ids within the pages.
Add the same navigation links on every page so the user can move easily between them.
Learning Outcome:
By the end of this assignment, you will have a basic multi-page website where all pages are linked together using HTML anchors. This will help you practice page navigation, linking, and structuring web content.
